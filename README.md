# Desafio Técnico em Java - Cadastro e Consulta de abastecimentos

* ### Objetivo
Desenvolver uma aplicação simples em Java para cadastro e consulta de abastecimentos em um posto
de combustível, com armazenamento em banco de dados e exibição dos dados via Java Swing ou API
REST.

* ### Funcionalidades
CRUD de Tipos de Combustível.

CRUD de Bombas de Combustível (relacionadas a um tipo
de combustível).

CRUD de Abastecimentos (com data, volume abastecido e
valor total).

Persistência de dados (PostgreSQL).
Consulta de todos os dados via API.

- Foi implementado um padrão de arquitetura em camadas, DTOs, documentação com o Swagger e clean code.
